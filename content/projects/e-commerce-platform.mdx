## The Challenge

HTWK Leipzig's existing e-learning platform was showing its age — poor mobile experience, brittle authentication, and a UI that discouraged engagement. Students avoided it. Support requests piled up. The university needed a ground-up rebuild that could scale with growing demand.

## What We Built

A modern Spring Boot/Kotlin backend paired with a responsive Tailwind CSS frontend. The platform features comprehensive language learning tools — C-tests with real-time feedback, spaced-repetition flashcard trainers, and progress tracking across courses.

Authentication was a key challenge: the system integrates SAML2 for university SSO alongside OAuth2 for external access, giving students seamless login regardless of context.

## The Result

Since launch, student engagement has increased by 60%. Technical support requests dropped by half. The platform handles peak-semester load without degradation and has become a model for future HTWK digital services.
