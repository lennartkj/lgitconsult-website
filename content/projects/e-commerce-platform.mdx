## The Challenge

 HTWK Leipzig needed to replace their outdated e-learning system that was struggling with performance issues and lacked modern features expected in today's educational technology landscape. The legacy platform couldn't support the growing number of courses and students, and offered a poor user experience that hindered learning outcomes.

## Our Solution

We built a robust Spring Boot/Kotlin application with a modern frontend, using a layered architecture designed specifically for educational contexts:

- **Spring Boot/Kotlin Backend**: Scalable, maintainable API services with excellent performance
- **REST API Architecture**: Clean separation of concerns with standardized endpoints
- **Tailwind CSS Frontend**: Responsive, accessible interface that works across all devices
- **Multi-Authentication System**: Support for SAML2, OAuth2, and traditional authentication
- **Comprehensive Learning Tools**: C-tests, flashcard trainers, and terminology management
- **Advanced Analytics**: Detailed insights for teachers to track student progress
- **Docker Deployment**: Containerized deployment with monitoring and backup solutions

## Key Features

### Course Management
A flexible system for creating and organizing educational content into courses and units, with support for various content types and learning paths.

### Learning Tools
Specialized educational tools including C-tests for language proficiency assessment, flashcard trainers for vocabulary practice, and terminology management.

### Teacher Analytics
Comprehensive analytics for instructors to monitor group performance, course completion rates, skill proficiency, and individual student achievements.

### Multi-Authentication Support
Secure authentication with multiple methods including SAML2 for institutional identity providers, OAuth2 for social login, and traditional username/password authentication.

## The Results

The new platform has transformed the digital learning experience at HTWK Leipzig, with significant improvements:

- 60% increase in student engagement with course materials
- 50% reduction in technical support requests
- 70% faster course creation and management for instructors
- Seamless integration with existing university systems
- Improved accessibility compliance, making learning accessible to all students

## Technologies Used

- **Backend**: Spring Boot 3.2, Kotlin 1.9
- **Database**: PostgreSQL 14
- **Security**: Spring Security with SAML2 and OAuth2
- **Frontend**: Tailwind CSS
- **Monitoring**: Prometheus, Grafana, Micrometer
- **Resilience**: Resilience4j, Caffeine caching
- **Testing**: JUnit 5, MockK, Kotest, Testcontainers
- **Deployment**: Docker, Docker Compose
